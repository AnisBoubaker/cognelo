# Plugin: Placeholder

This README is for the placeholder plugin only.

The placeholder plugin exists as a minimal activity shell while a real pedagogical activity is still being designed.

## Purpose

`@cognara/plugin-placeholder` provides:

- a lightweight registered activity type
- localized labels
- a minimal plugin boundary example

## Activity Type

- `placeholder`

## Current State

This plugin intentionally relies only on core activity records and does not currently need plugin-owned persistence or custom routes.
